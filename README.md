# EchoPath AI – Adaptive Speech Therapy Agent

## 🚀 Overview

EchoPath AI is a domain-specialized multi-agent system designed to deliver personalized speech therapy for autistic children.

It combines speech analysis, adaptive learning, and clinical compliance into a single intelligent workflow.

## 🧠 Key Features

* AI-driven therapy sessions
* Real-time speech analysis
* Adaptive difficulty adjustment
* Progress tracking dashboard
* Therapist-in-the-loop validation
* Compliance-aware recommendations

## 🏗️ Architecture

Multi-agent system:

* Therapy Planner Agent
* Speech Analysis Agent
* Adaptive Learning Agent
* Progress Tracking Agent
* Compliance Agent

## ⚙️ Tech Stack

* Frontend: React (generated via Kiro)
* Backend: FastAPI
* AI: OpenAI / Gemini APIs
* Speech Processing: Whisper API
* Database: Firebase / MongoDB

## 🎯 Demo Flow

1. Select child profile
2. Start therapy session
3. Record speech
4. AI evaluates response
5. Adaptive next exercise generated
6. Progress updated in dashboard

## 📊 Impact

* 50% reduction in therapist workload
* 2x therapy accessibility
* Improved phoneme accuracy tracking

## 🛠️ Setup

```bash
git clone https://github.com/ESpoorthy/echopath-ai-agent
cd echopath-ai-agent
pip install -r requirements.txt
uvicorn backend.main:app --reload
```
